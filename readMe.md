<h2>Aplicação web para gerenciamento de pedidos de uma cafeteria</h2>

![](https://img.shields.io/badge/javascript-informational?style=flat&logo=javascript&logoColor=white&color=gold)
![](https://img.shields.io/badge/HTML-informational?style=flat&logo=html5&logoColor=white&color=blue)
![](https://img.shields.io/badge/CSS-informational?style=flat&logo=css3&logoColor=white&color=pink)

> Sistema web responsivo para treino de habilidades de front-end.
> Site responsivo, que obtém dados de um arquivo json. Para acessar mais itens, basta atualizar o arquivo json.
> Funcionalidades de carrinho e de adição e exclusão de itens totalmente funcionais.
